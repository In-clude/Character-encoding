# 简单的字符编码转换
## 用于解决部分MMD模型ReadMe文档的字符乱码的问题

![程序截图](https://github.com/In-clude/Character-encoding/blob/master/%E6%88%AA%E5%9B%BE/%E7%A8%8B%E5%BA%8F%E6%88%AA%E5%9B%BE0.jpg?raw=true)

 Unicode为程序默认编码
 
 日文编码：“shift_jis”
 中文编码：“gb2312”

 使用:
 将文件拖入多选框即可读取,也可粘贴文件地址进行读取
